# Ultimate Vim Config
<p align="center"> Just paste and run vim, no need for plugin instalation </p>

### Theme
- gruvbox

### Keybindings
- ` `:            leader
- `<CR>`:         replace word (ciw vim default)
- `<leader>f`:    fold line
- `ff`:           open buffers
- `<C-f>`:        open file tree
- `<leader>t`:    open terminal
- `<leader>hlkj`: switch to window at h, l, k, j
